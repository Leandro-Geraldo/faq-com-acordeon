## Criar um acordeon
### Desafio
- ao clicar em uma pergunta deve acontecer
uma animação de acordeon para abrir o item
e mostrar o texto escondido, e a seta deve
mudar de cor.
- ao clicar em um item para abri-lo todos os
outros itens devem ser fechados.
